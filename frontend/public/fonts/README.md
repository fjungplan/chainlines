# Self-Hosted Montserrat Fonts

This directory contains self-hosted Montserrat font files for GDPR compliance.

## Files

- `Montserrat-Regular.woff2` / `Montserrat-Regular.woff` - Regular (400) weight
- `Montserrat-SemiBold.woff2` / `Montserrat-SemiBold.woff` - SemiBold (600) weight
- `Montserrat-Bold.woff2` / `Montserrat-Bold.woff` - Bold (700) weight

## Download

Montserrat fonts can be downloaded from:
- GitHub: https://github.com/JulietaUlanova/Montserrat/releases
- Google Fonts: https://fonts.google.com/specimen/Montserrat

## GDPR Compliance

These fonts are self-hosted (not served from Google Fonts CDN) to ensure compliance with GDPR regulations (German DSGVO). This prevents automatic IP address transmission to Google's servers.

See `/src/index.css` for @font-face declarations and font usage across the site.
